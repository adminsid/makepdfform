# MakePDFForm

MakePDFForm is a powerful, web-based PDF creation and manipulation engine designed for speed and simplicity. 

## Key Features

- **Robust Form Engine**: Utilize drag-and-drop tools to construct highly detailed layout forms with rich inputs like text fields, checkboxes, and signatures.
- **Secure Authentication**: Built on top of `better-auth`, it supports Google, GitHub, and passwordless authentication strategies, ensuring bullet-proof data partitioning.
- **Isolated User Dashboards**: Forms and submissions belong strictly to the authenticated user. View, share, and track all form variants from your custom dashboard securely.
- **Lightning-fast Submissions**: Form responses are securely submitted natively utilizing high-throughput Edge APIs, ensuring maximum availability.
- **Monetization & Plans**: Includes a premium tier structure backed by Polar/Stripe, allowing users to unlock unlimited forms, advanced submissions features, and custom branding constraints via a streamlined frontend.
- **Cloudflare Integration**: Built entirely on Cloudflare Workers, Pages, D1 database, and R2 Object Storage for an unbeatably crisp distributed architecture globally.
